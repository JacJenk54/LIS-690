# Omeka

## Prerequisites
1. Install ImageMagick
- `sudo apt install imagemagick`
2. Enable Apache `mod_rewrite`
- `sudo a2enmod rewrite`
3. Restate Apache
- `sudo systemctl restart apache2`

1. Use `wget` to install Omeka
